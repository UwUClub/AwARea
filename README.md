### SSH

```bash
git clone git@github.com:UwUClub/AwArea.git
```

## Using the application
### Server
First start the server
```bash
cd back && npm install && npm start
```
### Client
Then start the client
```bash
cd flutter_area && flutter pub get && flutter run
```
